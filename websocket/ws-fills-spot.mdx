# Fills Spot

Spot fills stream uses the **multiplex WebSocket endpoint**:

```text
wss://api-eu.hypedexer.com/ws
```

All subscriptions go through `/ws`.\
Authenticate using `X-API-Key` during the handshake.

---

## 1. Subscribe to spot fills

### Client message

```json
{
  "method": "subscribe",
  "subscription": {
    "type": "fills_spot"
  }
}
```
