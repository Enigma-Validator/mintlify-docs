---
title: "Features"
---

HypeDexer focuses on **fills** (executions) as the source of truth for building higher-level analytics (P&L, volume, liquidation/win-rate KPIs, etc.). Historical data is backfilled; live data is continuously ingested. 

- **Live data:** Latest indexed fills and related “now” views.
- **Historical data:** Backfilled datasets suitable for aggregates and time-series analyses.
