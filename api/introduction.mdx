# Introduction

Welcome to the **Hypedexer API** — your gateway to real-time on-chain data and analytics.  
Our API allows developers, analysts, and dApps to access blockchain data with high performance and reliability.

---

### 🚀 Free Plan

We offer a **free plan** so you can start testing our API right away.  
You can create an API key in your [dashboard](https://www.app.hypedexer.com/) and start exploring all available endpoints.

> ⚠️ Please note:  
> The **free plan** is designed for experimentation and testing.  
> Requests made with free-tier keys may experience **higher latency** compared to paid plans.

---

### 💡 Why use Hypedexer API?

- Access to real-time blockchain data.  
- Simple and secure authentication with API keys.  
- Multiple regional endpoints (Japan and Europe).  
- Easy integration for developers building dashboards, bots, or analytics tools.
