# Live data

Reproduced Hyperliquid endpoints served through our own infrastructure for stable, low-latency access — ideal for market makers and latency-sensitive applications.

**REST** is served via `POST /info` on the API host with typed JSON bodies (`metaAndAssetCtxs`, `allMids`, fills, order books, proxied HL types, etc.). **WebSocket** streams are available via `wss://api.hypedexer.com/ws?mode=mirror`.

For **indexed** queries use the **Indexed Data** tab.
