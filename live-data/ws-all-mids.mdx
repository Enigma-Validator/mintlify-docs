Subscribe with `{"method":"subscribe","subscription":{"type":"allMids"}}` (optional `dex`). Acknowledgement uses **`subscriptionResponse`** (see **`SubscriptionResponseAck`**). Snapshots use **`AllMidsSnapshot`**; changes use **`AllMidsUpdate`**.
