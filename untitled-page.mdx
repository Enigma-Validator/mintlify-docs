# Hyperliquid REST API Access

You can use the following REST endpoint to access the **Hyperliquid API** without any rate limits or restrictions.

```
https://api.hypedexer.com/info
```

This API provides access to all official **Hyperliquid info endpoints** while removing the usual API limitations.

## Features

- No rate limits
- No request quotas
- Full access to Hyperliquid info endpoints
- Simple REST interface
- Compatible with the official Hyperliquid request format

## Authentication

Requests require an API key using the `X-API-Key` header.

Example:

```
X-API-Key: YOUR_API_KEY
```

## Example Request

```bash
curl -X POST https://api.hypedexer.com/info \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{"type": "spotMetaAndAssetCtxs"}'
```

## Request Format

All requests follow the same format as the official Hyperliquid API.

Example body:

```json
{
  "type": "meta"
}
```

### Fields

| Field | Description |
|------|-------------|
| `type` | The Hyperliquid endpoint you want to query |

## Example Response

The API returns the same response format as the official Hyperliquid API.

```json
{
  "assets": [...],
  "universe": [...]
}
```

## Notes

- This API is designed to be fully compatible with existing Hyperliquid integrations.
- Responses maintain the exact same structure as the official API.
